# EchoGraph-Audio-Intelligence-System
EchoGraph is an audio-based intelligence system that classifies environmental sounds and detects anomalies using machine learning.

## Features
- Audio classification  
- Feature extraction using MFCC  
- Anomaly detection  

## Tech Stack
Python, Librosa, TensorFlow  

## Run
streamlit run app.py

## How It Works
- Extracts audio features  
- Uses ML model for classification  
- Outputs environment type  

## Future Improvements
- Multi-class classification  
- Real-time audio streaming  
- Deep learning models
